# BBS

## 소개
* JSP를 이용하여 게시판 서비스 구현   
* 기간 : 2021.07.01 ~ 2021.07.05

## 개발 환경 및 기술
* JAVA JDK 11
* JAVA, HTML, CSS, Javascript
* Bootstrap
* MySql

## 구현 기능
* 메인 페이지 소개
* 회원가입, 로그인, 로그아웃
* 게시판 : 글 목록, 글 쓰기, 글 수정, 글 삭제, 페이징 기능 (쓰기, 수정, 삭제는 회원만 가능)
